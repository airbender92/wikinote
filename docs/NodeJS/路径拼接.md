
# `const docPath = path.join(process.cwd(), CONFIG.docDir);`

## 一句话解释
**这行代码的作用：拼接出一个「绝对路径」，指向你项目里的文档目录。**

---

## 逐段拆解（最关键）
### 1. `path.join()`
- Node.js 内置的**路径拼接工具**
- 自动帮你处理 `/` `\` 斜杠问题（Windows / Mac / Linux 通用）
- 不会出现路径错误

### 2. `process.cwd()`
- **当前执行 Node 命令的工作目录**
- 99% 情况下 = **你的项目根目录**
- 绝对路径，例如：`/Users/xxx/project`

### 3. `CONFIG.docDir`
- 你项目配置里的**文档文件夹名称**
- 例如：`'docs'` 或 `'documents'`

### 4. 最终结果
```
项目根目录 + 文档目录 = 文档的绝对路径
```

---

## 举个真实例子
假设：
- 项目根目录：`/my-project/`
- `CONFIG.docDir = 'docs'`

那么：
```javascript
const docPath = path.join(process.cwd(), CONFIG.docDir);

// docPath 等于
// /my-project/docs
```

---

## 这行代码通常用来做什么？
- 读取项目里的 **markdown 文档**
- 读取静态资源
- 生成文档网站（VitePress / VuePress / Docsify）
- 遍历目录文件

---

## 完整可运行代码（你直接复制就能用）
```javascript
const path = require('path');
const CONFIG = {
  docDir: 'docs' // 你的文档目录
};

// 拼接文档绝对路径
const docPath = path.join(process.cwd(), CONFIG.docDir);

console.log('文档目录路径：', docPath);
```

---

## 关键点总结
- `path.join()` → **安全拼接路径**
- `process.cwd()` → **项目根目录**
- `CONFIG.docDir` → **文档文件夹名**
- `docPath` → **最终文档目录的绝对路径**

你应该是打错啦，是 **`path.join()`**！

我直接用**最简单、最清晰、不绕弯**的方式，把 **`path.join` 和 `path.resolve` 一起讲透**，你马上就能分清！

# 一句话解释
**`path.join()` = 把路径片段**简单拼接**成一个完整路径（不一定是绝对路径）**

---

# 核心区别（最重要）
| 方法 | 作用 | 结果 |
|------|------|------|
| **path.join** | 单纯拼接路径 | 可相对、可绝对 |
| **path.resolve** | 从后往前拼，**一定返回绝对路径** | 永远是绝对路径 |

---

# 超级简单例子
```javascript
const path = require('path');

// 1. path.join → 单纯拼接
path.join('/a', 'b', 'c'); // 结果：/a/b/c
path.join('a', 'b');       // 结果：a/b （相对路径）

// 2. path.resolve → 一定拼出绝对路径
path.resolve('a', 'b');    // 结果：/当前项目目录/a/b （绝对路径）
```

---

# 最常用场景
### 1. **path.join**
适合**单纯拼接路径**，不想变成绝对路径的时候：
```js
path.join('dist', 'js', 'app.js')
// 结果：dist/js/app.js
```

### 2. **path.resolve**
适合**必须要绝对路径**的时候（webpack 配置、文件读取）：
```js
path.resolve(__dirname, 'dist')
// 结果：/你项目的目录/dist
```

---

# 终极口诀（记这个就够）
- **path.join**：**老老实实拼接**
- **path.resolve**：**拼完一定是绝对路径**

---

### 总结
`path.join` = **路径拼接工具**
`path.resolve` = **路径拼接 + 转绝对路径**
