# 高级媒体流媒体（Sophisticated Media Streaming）
## 人话精简总结

这一章讲的是：**用 Nginx 做视频、音频流媒体服务**，主要支持两种格式：
- **MP4（MPEG-4）**
- **FLV（Flash Video）**

Nginx 本身非常适合做**高并发、大流量**的媒体分发，这一章会教你怎么配置。

---

## 核心要点

### 1. Nginx 原生支持的流媒体能力
- **MP4 伪流媒体**
  支持断点续传、拖动进度条播放（`mp4` 模块）。
- **FLV 流媒体**
  用于 Flash 视频流（现在用得少，但仍有大量系统在用）。
- **带宽限制**
  控制单用户下载速度，防止服务器被跑满。

### 2. Nginx Plus 增强功能（收费版）
- **HLS（HTTP Live Streaming）**
  实时切片，适配手机、iOS、Android 直播/点播。
- **HDS（HTTP Dynamic Streaming）**
  动态流媒体，适配已切片的媒体文件。
- **高级码率控制（bitrate limiting）**
  更精细控制流媒体速度，保证更多用户同时观看不卡顿。

### 3. 为什么用 Nginx 做流媒体？
- **超高并发**，能扛大量用户同时看视频
- **占用资源极低**
- **支持进度条拖动播放**
- 适合做：短视频、点播、直播回放点播、大文件下载
