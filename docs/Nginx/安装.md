# 二、你只需要 2 条命令，就能启动 NGINX
打开你的 **Windows 命令行（CMD/PowerShell）**，直接运行：

## 1. 拉取官方 NGINX 镜像
```bash
docker pull nginx
```

## 2. 启动 NGINX
```bash
docker run -d -p 80:80 --name my-nginx nginx
```

然后打开浏览器访问：
```
http://localhost
```
看到 NGINX 欢迎页 = **环境准备完成 ✅**

# 三、怎么用 Docker 学这本书？
这本书所有配置文件都在这里：
```
/etc/nginx/
```

你在 Docker 里直接编辑就行！

## 进入 NGINX 容器内部（像进入 Linux 一样）
```bash
docker exec -it my-nginx /bin/bash
```

进去后，你就可以：
```bash
# 编辑配置
vi /etc/nginx/nginx.conf

# 测试配置
nginx -t

# 重启
nginx -s reload
```

**和书里的命令一模一样！**
