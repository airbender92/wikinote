# Problem: You need to serve static content with NGINX.

# Solution: 

**把 NGINX 默认的网站配置文件（`/etc/nginx/conf.d/default.conf`）清空，替换成我给你的这段新配置。**
```nginx
server { 
  listen 80 default_server; 
  server_name www.example.com;
  location / { 
    root /usr/share/nginx/html; 
    # alias /usr/share/nginx/html;
    index index.html index.htm;
  }
}
```

这段配置的作用：
- 监听 **80 端口**（网页默认端口）
- 设置网站域名：`www.example.com`
- 指定网站文件存放目录：`/usr/share/nginx/html`
- 默认打开首页：`index.html`

---

# 对你（Docker 环境）的操作步骤
你只需要做 2 件事：
1. 打开这个默认配置文件
2. 把里面内容**全删掉**，粘贴这段新代码

## 命令（直接照着输）
```bash
# 编辑默认配置文件
vi /etc/nginx/conf.d/default.conf
```

## 编辑方法：
1. 按 `i` 进入编辑模式
2. **把原有内容全部删除**
3. 粘贴这段代码：
```nginx
server {
  listen 80 default_server;
  server_name www.example.com;

  location / {
    root /usr/share/nginx/html;
    index index.html index.htm;
  }
}
```
4. 按 `Esc`，输入 `:wq` 保存退出

## 最后让配置生效
```bash
nginx -t && nginx -s reload
```
