# 7.13 根据**国家/地区**限制访问（GeoIP 模块）
我给你用**最简单、最直白、一看就会**的方式讲👇

---

# 核心一句话
**安装 GeoIP 模块 → 让 Nginx 能识别用户来自哪个国家 → 允许/禁止某些国家访问**

比如：
**只允许美国、加拿大访问，其他国家全部 403 禁止**

---

# 完整流程（超级清晰）
## 1. 安装模块
```bash
apt install nginx-module-geoip
```

## 2. 加载模块
```nginx
load_module "/etc/nginx/modules/ngx_http_geoip_module.so";
```

## 3. 配置规则：允许哪些国家
```nginx
map $geoip_country_code $country_access {
    default 1;  # 其他国家 → 1（禁止）
    "US"  0;    # 美国 → 0（允许）
    "CA"  0;    # 加拿大 → 0（允许）
}
```

意思：
- `0` = 允许
- `1` = 拒绝

## 4. 在 server 里拦截
```nginx
if ($country_access = '1') {
    return 403;
}
```

---

# 逐句翻译成人话
```nginx
map $geoip_country_code $country_access {
    default 1;
    "US" 0;
    "CA" 0;
}
```

→ **如果用户来自 US/CA → 允许
其他所有国家 → 禁止**

```nginx
if ($country_access = '1') {
    return 403;
}
```

→ **如果是禁止国家 → 直接返回 403**

---

# 变量从哪来？
```
$geoip_country_code
```
是 GeoIP 模块**自动提供**的，值是国家二字码：
`US`、`CN`、`JP`、`KR`、`GB`、`DE` 等

---

# 你可以轻松改成：
## ✅ 只允许中国访问
```nginx
map $geoip_country_code $allow_country {
    default 1;
    "CN" 0;
}
```

## ✅ 禁止某些国家
```nginx
map $geoip_country_code $deny_country {
    default 0;
    "RU" 1;
    "IR" 1;
}
```
