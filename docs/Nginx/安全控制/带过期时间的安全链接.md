# 7.8 带**过期时间**的安全链接（终极防盗链）
我给你用**最简单、最实用、一看就懂**的方式讲👇

---

# 核心一句话
**这是上一节的升级版：
不仅需要正确签名，链接还**会过期**，
而且只能**同一个IP**使用，别人拿到链接也用不了。**

---

# 这个配置实现了 3 大超强安全功能
1. **URL 带过期时间**（超时自动失效）
2. **绑定用户 IP**（别人偷了链接也用不了）
3. **MD5 签名验证**（防篡改）

---

# 先看配置（我给你逐行翻译）
```nginx
location /resources {
    root /var/www;

    # 从 URL 里取两个参数：md5签名、过期时间
    secure_link $arg_md5,$arg_expires;

    # 拼接加密字符串：过期时间 + 路径 + 客户端IP + 密钥
    secure_link_md5 "$secure_link_expires$uri$remote_addrmySecret";

    # 签名错误 → 403 禁止
    if ($secure_link = "") { return 403; }

    # 签名正确，但已过期 → 410 失效
    if ($secure_link = "0") { return 410; }
}
```

---

# 1. 这个链接长什么样？
```
https://your.com/resources/file.mp4?md5=XXX&expires=1735689600
```

包含 2 个参数：
- `md5`：签名
- `expires`：过期时间（Unix 时间戳）

---

# 2. 加密规则是什么？
```
md5( 过期时间 + 文件路径 + 客户端IP + 密钥 )
```

例子：
```
md5("1735689600/resources/file.mp41.2.3.4mySecret")
```

---

# 3. 为什么这么安全？
### ① 链接**会过期**
超过 `expires` 时间 → **自动失效**

### ② 链接**绑定 IP**
换个IP访问 → **直接无效**
别人偷了链接也用不了

### ③ 密钥只有你知道
无法伪造链接

---

# 4. 两个关键判断
### `$secure_link = ""`
→ **签名错误/被篡改**
→ 返回 403 Forbidden

### `$secure_link = "0"`
→ **签名正确，但链接超时了**
→ 返回 410 Gone（永久失效）

---

# 5. 410 Gone 是什么？
就是告诉浏览器：
**这个链接已经永久过期，不要再访问了**
比 404 更准确。

---

# 6. 这个配置用在什么场景？
- 视频付费观看
- 私有文件临时下载
- 会员专属资源
- 防止链接被转发、盗用
- 合规要求（链接不能永久有效）
