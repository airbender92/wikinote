# Health Checks（健康检查）核心翻译+极简总结

## 翻译
在复杂网络架构中，负载均衡器**最重要的功能之一**，就是通过**周期性健康检查**持续监控后端服务器状态。

健康检查确保负载均衡器**只把流量转发给可用、状态正常的服务器**，从而保证应用的**高可用**与最佳性能。

就像定期体检监测人的健康、及早发现潜在问题一样，计算机网络中的健康检查包括一系列主动探测手段：
- Ping 检测
- 延迟测试
- 带宽测试

用于评估系统或网络的整体健康状况与性能。

此外，健康检查还能在问题恶化前提前发现并定位故障，支持**主动维护与排障**。

---

# 极简一句话总结
## **健康检查 = 负载均衡器定期“体检”后端服务器**
- 只给健康的服务器发流量
- 自动剔除宕机/异常节点
- 提高可用性，减少用户报错
