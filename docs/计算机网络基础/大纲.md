# 计算机网络基础 - 全知识点总结

## 一、计算机网络概述
1. **什么是计算机网络**
   - 多台计算机通过通信设备和线路互联，实现资源共享和数据交换
2. **网络的主要功能**
   - 数据通信
   - 资源共享（硬件、软件、数据）
   - 分布式处理
3. **网络分类**
   - 按覆盖范围：
     - LAN 局域网
     - MAN 城域网
     - WAN 广域网
   - 按拓扑结构：
     - 星型、总线型、环型、树型、网状
4. **网络性能指标**
   - 速率（bps）
   - 带宽
   - 时延（发送、传播、处理、排队）
   - 吞吐量
   - 丢包率

---

## 二、网络体系结构与协议
1. **协议三要素**
   - 语法、语义、时序
2. **分层思想的好处**
   - 独立性强、易维护、易扩展、标准化
3. **OSI 七层模型（必须背）**
   1. 应用层
   2. 表示层
   3. 会话层
   4. 传输层
   5. 网络层
   6. 数据链路层
   7. 物理层
4. **TCP/IP 四层模型（实际使用）**
   - 应用层
   - 传输层
   - 网际层
   - 网络接口层
5. **数据封装与解封装**
   - 每一层加头部/尾部
   - 从上到下封装，从下到上解封装

---

## 三、物理层
1. **作用**
   - 传输比特流 0/1
2. **传输介质**
   - 双绞线（直通线/交叉线）
   - 光纤（单模/多模）
   - 无线电波、微波、红外线
3. **通信方式**
   - 单工、半双工、全双工
4. **数据传输方式**
   - 串行 / 并行
   - 同步 / 异步
5. **信道复用技术**
   - FDM 频分复用
   - TDM 时分复用
   - WDM 波分复用
   - CDMA 码分多址

---

## 四、数据链路层
1. **作用**
   - 组帧、差错控制、流量控制、介质访问控制
2. **成帧方法**
   - 字符计数法
   - 首尾字符法
   - 零比特填充
3. **差错控制**
   - 检错：CRC 循环冗余校验
   - 纠错：海明码
4. **流量控制**
   - 停止等待协议
   - 滑动窗口（GBN、SR）
5. **局域网技术**
   - Ethernet 以太网
   - CSMA/CD 载波监听多路访问/冲突检测
6. **设备**
   - 网卡、交换机、网桥

---

## 五、网络层
1. **作用**
   - 路由选择、分组转发、拥塞控制
2. **IP 协议**
   - IPv4 地址格式
   - A/B/C/D/E 类地址
   - 公网 IP / 私网 IP
3. **子网划分与超网**
   - 子网掩码
   - CIDR 无类别域间路由
4. **ARP 与 RARP**
   - ARP：IP → MAC
   - RARP：MAC → IP
5. **ICMP 互联网控制报文协议**
   - ping
   - tracert
6. **路由协议**
   - 内部：RIP、OSPF
   - 外部：BGP
7. **设备**
   - 路由器

---

## 六、传输层
1. **作用**
   - 端到端通信、可靠传输、复用分用
2. **端口号**
   - 知名端口 0~1023
   - 注册端口 1024~49151
   - 动态端口 49152~65535
3. **UDP 协议**
   - 无连接、不可靠、快速、开销小
   - 适用：直播、DNS、视频通话
4. **TCP 协议**
   - 面向连接、可靠、字节流
5. **TCP 三次握手**
   - 建立连接过程
6. **TCP 四次挥手**
   - 断开连接过程
7. **TCP 可靠传输机制**
   - 确认应答 ACK
   - 超时重传
   - 流量控制（滑动窗口）
   - 拥塞控制（慢启动、拥塞避免、快重传、快恢复）

---

## 七、应用层
1. **常见应用层协议**
   - HTTP/HTTPS
   - DNS 域名系统
   - FTP 文件传输
   - SMTP/POP3/IMAP 邮件
   - DHCP 自动获取 IP
   - Telnet / SSH
2. **HTTP 协议重点**
   - 请求方法：GET、POST、PUT、DELETE
   - 状态码：1xx、2xx、3xx、4xx、5xx
   - 请求头、响应头、Cookie、Session
3. **HTTPS = HTTP + TLS/SSL**
   - 加密、身份验证、数据完整性
4. **DNS 解析过程**
   - 浏览器缓存 → 系统缓存 → 路由器缓存 → 递归 DNS → 根域名服务器

## 九、Web 通信流程（综合题必考）
1. 浏览器输入 URL
2. DNS 解析域名 → IP
3. 建立 TCP 连接（三次握手）
4. 发送 HTTP 请求
5. 服务器处理并返回响应
6. 浏览器渲染页面
7. 断开 TCP 连接（四次挥手）
