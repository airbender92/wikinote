# 物理层（核心知识点 · 必背）

## 1. 作用
- 传输**比特流 0/1**
- 只负责**传输原始的二进制位**，不关心内容是什么
- 定义电气、机械、功能、规程特性

一句话：
**物理层只管传 0 和 1，不管数据含义。**

---

## 2. 核心关键词
- 传输介质
- 接口标准
- 电平、光信号、无线电波
- 比特同步
- 不处理差错、不识别协议

---

## 3. 典型设备
- 网线（双绞线）
- 光纤
- 集线器（Hub）
- 网卡（部分功能）
- 中继器

---

## 4. 一句话总结
**物理层：最底层，只传 01 比特流，不理解数据。**
