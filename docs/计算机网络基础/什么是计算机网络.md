# 什么是计算机网络（简洁标准定义）
**计算机网络，就是把地理位置不同、具有独立功能的多台计算机，通过通信设备和传输介质互联起来，在网络软件（协议、操作系统）的支持下，实现数据通信和资源共享的系统。**

简单一句话：
**多台电脑连在一起，能互相传数据、共享东西，就是计算机网络。**

---

# 用最通俗的话解释
- 你家里的电脑、手机、平板连同一个 Wi-Fi → 就是一个**小型局域网**
- 全世界无数个这样的小网络互相连接 → 就组成了**互联网（Internet）**

它的核心就两件事：
1. **互相通信**：发消息、传文件、刷网页、看视频
2. **资源共享**：共享打印机、共享文件、共享服务器数据

---

# 三个必备组成部分
1. **主机（计算机）**
   手机、电脑、服务器等，是数据的发送和接收方。

2. **通信设备**
   路由器、交换机、网卡、基站、光纤等，负责“铺路”和“转发”。

3. **通信协议（规则）**
   比如 TCP/IP、HTTP、Wi-Fi 协议。
   相当于大家共同遵守的“语言”，否则设备之间无法交流。

---

# 计算机网络的主要功能
1. **数据交换**：聊天、发邮件、传文件、刷视频
2. **资源共享**：共享硬盘、打印机、软件、数据库
3. **分布式处理**：多台电脑一起完成一个大任务（如云计算）
4. **提高可靠性**：一台机器挂了，其他可以顶上

---

# 最简单例子
- 你用微信发消息 → 走计算机网络
- 打开浏览器看网页 → 走计算机网络
- 手机和电脑互传文件 → 走计算机网络
