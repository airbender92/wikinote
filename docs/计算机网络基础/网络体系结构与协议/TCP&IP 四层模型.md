# TCP/IP 四层模型（实际使用版 · 必背）
这是**互联网真正在用的模型**，和 OSI 七层对应，但更简洁实用。

---

## 4. 应用层
（对应 OSI：应用层 + 表示层 + 会话层）
- 作用：给应用程序提供服务，直接面向用户
- 协议：HTTP、HTTPS、DNS、FTP、SMTP、SSH
- 一句话：**用户能感知到的网络功能都在这**

---

## 3. 传输层
（和 OSI 传输层基本一致）
- 作用：端到端的数据传输，进程间通信
- 协议：**TCP、UDP**
- 关键词：端口号、可靠传输、流量控制

---

## 2. 网际层（网络层）
（对应 OSI 网络层）
- 作用：路由选择、IP 寻址、数据包转发
- 协议：**IP、ARP、ICMP（ping）**
- 关键词：IP 地址、找路、路由器

---

## 1. 网络接口层
（对应 OSI：数据链路层 + 物理层）
- 作用：负责与物理网络接口，传输帧、比特流
- 设备：网卡、交换机、网线、光纤
- 关键词：MAC 地址、01 信号、物理传输

---

# 四层速记
**应 → 传 → 网 → 接口**

---

# OSI 七层 ↔ TCP/IP 四层 对应关系
- OSI 7、6、5 层 → TCP/IP **应用层**
- OSI 4 层 → TCP/IP **传输层**
- OSI 3 层 → TCP/IP **网际层**
- OSI 2、1 层 → TCP/IP **网络接口层**

---

# 一句话总结
- **OSI 七层**：理论标准，用来考试
- **TCP/IP 四层**：现实标准，互联网真在用
