# OSI 七层模型（必背版 + 通俗讲解）
我给你按**从上到下**顺序，每层一句话功能+记忆点，**考试、面试、记笔记直接用**。

## 7. 应用层（Application Layer）
- **作用**：给应用程序提供服务，直接和用户交互
- **协议**：HTTP、HTTPS、FTP、DNS、SMTP
- **记忆**：用户用的软件都在这一层

## 6. 表示层（Presentation Layer）
- **作用**：数据格式转换、加密、解密、压缩、解压
- **记忆**：翻译官 + 加密解密

## 5. 会话层（Session Layer）
- **作用**：建立、管理、终止会话（连接）
- **记忆**：负责“通话”的建立与挂断

## 4. 传输层（Transport Layer）
- **作用**：端到端传输，可靠/不可靠传递
- **协议**：TCP、UDP
- **记忆**：负责数据可靠送达，端口号

## 3. 网络层（Network Layer）
- **作用**：路由选择，IP寻址，找路径
- **协议**：IP、ARP、ICMP
- **记忆**：负责找地址、选路线

## 2. 数据链路层（Data Link Layer）
- **作用**：组帧、差错检测、MAC寻址
- **设备**：交换机
- **记忆**：相邻节点之间传数据帧

## 1. 物理层（Physical Layer）
- **作用**：传输 0/1 比特流，电气特性
- **设备**：网线、光纤、集线器
- **记忆**：只传 0 和 1，不管内容

---

# 超级速记口诀（从上到下）
**应 表 会 传 网 链 物**

- 应：应用层
- 表：表示层
- 会：会话层
- 传：传输层
- 网：网络层
- 链：数据链路层
- 物：物理层

---

# 一句话串起来
用户用APP（应用层）→ 翻译加密（表示层）→ 建立会话（会话层）→ 端到端传输（传输层）→ 找IP路由（网络层）→ 组帧传数据（链路层）→ 传01比特（物理层）。
