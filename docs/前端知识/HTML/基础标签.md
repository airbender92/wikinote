# HTML 基础标签

## 什么是HTML

HTML（HyperText Markup Language）是用于创建网页的标准标记语言。

## 常用基础标签

### 标题标签

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
```

### 段落标签

```html
<p>这是一个段落</p>
```

### 链接标签

```html
<a href="https://www.example.com">链接文本</a>
```

### 图片标签

```html
<img src="image.jpg" alt="图片描述">
```

### 列表标签

#### 无序列表

```html
<ul>
  <li>项目1</li>
  <li>项目2</li>
</ul>
```

#### 有序列表

```html
<ol>
  <li>项目1</li>
  <li>项目2</li>
</ol>
```
