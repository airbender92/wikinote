# CSS 选择器

## 什么是CSS选择器

CSS选择器是用于选择要应用样式的HTML元素的模式。

## 常用选择器

### 元素选择器

```css
p {
  color: red;
}
```

### 类选择器

```css
.class-name {
  font-size: 16px;
}
```

### ID选择器

```css
#element-id {
  background-color: blue;
}
```

### 后代选择器

```css
div p {
  margin: 10px;
}
```

### 子选择器

```css
div > p {
  padding: 5px;
}
```

### 相邻兄弟选择器

```css
h1 + p {
  font-weight: bold;
}
```

### 属性选择器

```css
input[type="text"] {
  border: 1px solid #ccc;
}
```
