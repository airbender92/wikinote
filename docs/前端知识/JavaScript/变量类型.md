# JavaScript 变量类型

## 变量声明

在JavaScript中，有三种声明变量的方式：

```javascript
var x = 10;      // 函数作用域
let y = 20;      // 块级作用域
const z = 30;    // 常量，不可修改
```

## 数据类型

### 基本数据类型

- **Number**：数字类型
  ```javascript
  let num = 100;
  let float = 3.14;
  ```

- **String**：字符串类型
  ```javascript
  let str = "Hello";
  let str2 = 'World';
  ```

- **Boolean**：布尔类型
  ```javascript
  let isTrue = true;
  let isFalse = false;
  ```

- **Null**：空值
  ```javascript
  let empty = null;
  ```

- **Undefined**：未定义
  ```javascript
  let undef;
  ```

- **Symbol**：唯一标识符
  ```javascript
  let sym = Symbol('id');
  ```

### 引用数据类型

- **Object**：对象
  ```javascript
  let obj = { name: "John", age: 30 };
  ```

- **Array**：数组
  ```javascript
  let arr = [1, 2, 3, 4, 5];
  ```

- **Function**：函数
  ```javascript
  function greet() {
    return "Hello";
  }
  ```
