## 二、查找真正运行的前端（3 个最实用命令）
### 1. 查找所有正在运行的前端进程（最准）
```bash
ps -ef | grep -E "node|npm|yarn|vite|vue|react"
```
如果有前端在运行，这里会显示：
- 进程名：`node` / `npm run dev`
- 项目目录：能直接看到代码在哪

### 2. 查找所有监听的 Web 端口（找前端端口）
```bash
ss -tlnp | grep -E "3000|5173|8080|9528|8081"
```
前端常用端口：
- **3000**（React）
- **5173**（Vite + Vue/React）
- **8080**（Webpack）
- **9528**（vue-element-admin）

### 3. 直接搜索服务器上所有前端项目目录
```bash
find / -name "package.json" 2>/dev/null | head -20
```
`package.json` 是前端项目**必带文件**，找到它就找到前端代码了。
