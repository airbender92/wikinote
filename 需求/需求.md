# 一、项目名称
**Markdown 知识点文档阅读系统**
（可自定义名称：学习笔记系统 / 知识库 / 文档中心）

# 二、项目目标
用于**个人/团队知识点记录、整理、阅读、复习**，支持多级菜单、文档导航、全文搜索，所有内容使用 Markdown 编写。

# 三、核心功能需求（你要的 4 点全覆盖）
## 1. Markdown 文档展示
- 所有文档使用 **Markdown 语法**编写
- 页面自动渲染 Markdown 为美观的阅读页面
- 支持：标题、列表、代码块、表格、图片、链接、加粗、斜体
- 阅读界面简洁、护眼、适合长时间学习

## 2. 多级菜单 + 子菜单 + 导航栏
- 左侧**固定导航菜单**（可收起/展开）
- 支持 **一级菜单 → 二级菜单 → 三级菜单**
- 点击菜单自动切换对应 Markdown 文档
- 当前打开文档自动高亮
- 顶部简单导航栏：显示项目名称、搜索框

## 3. 全文搜索功能
- 顶部/侧边提供**搜索框**
- 支持**标题 + 内容**全局搜索
- 输入关键词实时匹配
- 展示匹配结果，点击直接跳转到对应文档

## 4. 知识点记录与复习
- 文档按**分类/章节**组织
- 适合存放：学习笔记、技术知识点、面试总结、课程笔记
- 阅读体验轻量化、无干扰
- 支持快速跳转、翻页、上下篇切换

# 五、功能详细需求
## 1. 菜单管理（核心）
- 菜单配置文件化（无需数据库，直接 JSON/YAML）
- 支持：
  - 菜单名称
  - 菜单图标（可选）
  - 对应 Markdown 文件路径
  - 默认展开/折叠
- 示例结构：
```
前端知识
  ├─ HTML
  │   ├─ 基础标签.md
  │   └─ 表单元素.md
  ├─ CSS
  │   ├─ 选择器.md
  │   └─ 布局.md
  └─ JavaScript
      ├─ 变量类型.md
      └─ 作用域.md
```

## 2. Markdown 渲染
- 支持标准 Markdown
- 代码高亮（编程笔记必备）
- 支持目录自动生成（显示文档内小标题）
- 支持平滑滚动

## 3. 搜索功能
- 搜索范围：所有文档的**标题 + 内容**
- 不依赖后端，可做**前端本地搜索**
- 支持模糊匹配
- 回车/点击搜索，展示结果列表

## 4. 阅读体验优化
- 文档支持：上一篇 / 下一篇 跳转
- 自适应手机、电脑、平板
- 简洁护眼主题
- 菜单可展开/收起
